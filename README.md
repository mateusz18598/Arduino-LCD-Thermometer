# Arduino-LCD-Thermometer
 A simple thermometer with an I2C LCD display

**Required components:**
- Arduino UNO
- LCD I2C
- 4 Jumper wires - male to female jumper wires
- 3 Jumper wires - male
- 4.7K resistor
- DS18B20 temperature sensor

**Used libraries:**
- OneWire
- DallasTemperature
- LCD-I2C
